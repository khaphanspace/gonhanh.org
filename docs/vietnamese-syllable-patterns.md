## 1. Initial Consonants (C₁)

### 1.1 Single Initial Consonants (16)

```
b c d g h k l m n p q r s t v x
```

| Consonant | IPA | Example | Note |
|-----------|-----|---------|------|
| b | /ɓ/ | ba, bé, bỏ | Voiced stop |
| c | /k/ | ca, có, cơm | Before a, o, u, consonants |
| d | /z/~/j/ | da, để, dều | Voiced fricative |
| g | /ɣ/ | ga, góc, gọn | Before a, o, u; NOT before e |
| h | /h/ | ha, hoa, hóa | Voiceless fricative |
| k | /k/ | kéo, kè, kỳ | Before e, i, y; NOT before a, o, u |
| l | /l/ | la, lề, lời | Alveolar lateral |
| m | /m/ | ma, mệ, mía | Bilabial nasal |
| n | /n/ | na, né, nồi | Alveolar nasal |
| p | /p/ | pin, pè, pơi | Voiceless stop |
| q | /k/ | qua, quê, quí | Always followed by u |
| r | /z/~/r/ | ra, rề, rơi | Voiced fricative/trill |
| s | /s/ | sa, sề, sối | Voiceless fricative |
| t | /t/ | ta, tế, tôi | Voiceless stop |
| v | /v/ | va, về, vôi | Voiced fricative |
| x | /s/ | xa, xê, xương | Voiceless fricative |

### 1.2 Double Initial Consonants (11)

Digraphs representing single phonemes:

```
ch gh gi kh kr ng nh ph qu th tr
```

#### Aspiration Pairs (4)

| Pair | Single Equiv. | Example | Note |
|------|---------------|---------|------|
| ch | None (c+h) | cha, chế, chơi | Voiceless affricate |
| kh | None (k+h) | kha, khế, khối | Aspirated velar |
| ph | None (p+h) | pha, phế, phối | Aspirated bilabial |
| th | None (t+h) | tha, thế, thời | Aspirated alveolar |

#### Velar Variants (2)

| Pair | Contrast | Before Vowels | Example |
|------|----------|---------------|---------|
| g | /ɣ/ unaspirated | a, o, u | ga, gó, gụ |
| gh | /ɣ/ velar fricative | i, e, ê | ghi, ghế, ghê |

#### Affricate & Retroflex (2)

| Pair | Phoneme | Before Vowels | Example |
|------|---------|---------------|---------|
| gi | /z/~/j/ + i | (acts as initial) | gia, giề, giữ |
| tr | Retroflex | All vowels | tra, trế, trôi |

#### Nasal Clusters (2)

| Pair | Contrast | Before Vowels | Example |
|------|----------|---------------|---------|
| ng | /ŋ/ | a, o, u | nga, ngó, ngu |
| ngh | /ŋ/ velar | i, e, ê | nghi, nghề, nghê |

#### Labial + Glide (1)

| Pair | Always With | Example | Note |
|------|-------------|---------|------|
| qu | u (acts as glide) | qua, quê, quí | U is part of initial |

### 1.3 Triple Initial Consonant (1)

```
ngh - Only before i, e, ê
```

Example: `nghi, nghề, nghiêng`

### 1.4 No Initial Consonant (∅)

V pattern (vowel-initial syllables):

Examples: `a, ê, i, o, ơ, u, ư, y, ai, ao, ia, oa, ưa`

### 2.2 Valid Diphthongs (27)

Combinations of two vowels where first may be a glide.

#### A-Series (4)

```
ai ao au ay
```

| Diphthong | IPA | Ending | Example | Notes |
|-----------|-----|--------|---------|-------|
| ai | /aj/ | -i | hai, mai, tai | Falling to /i/ |
| ao | /aw/ | -o | cao, sao, nao | Falling to /w/ |
| au | /aw/ | -u | sau, mau, dâu | Falling to /w/ (spell variation) |
| ay | /aj/ | -y | hay, may, say | Falling to /j/ (spell variation) |

#### E-Series (2)

```
eo êu
```

| Diphthong | IPA | Type | Example | Notes |
|-----------|-----|------|---------|-------|
| eo | /ɛw/ | Falling | keo, theo, treo | e + o, ends in /w/ |
| êu | /ew/ | Falling | kêu, rêu, nêu | ê + u, ends in /w/ |

#### I-Series (3)

```
ia iê iu
```

| Diphthong | IPA | Type | Example | Notes |
|-----------|-----|------|---------|-------|
| ia | /iə/ | Rising | kia, mía, lía | i + a, ends in /ə/ |
| iê | /iə/ | Rising | tiên, kiên, liên | i + ê, ends in /ə/ |
| iu | /iw/ | Falling | dịu, kíu, tiu | i + u, ends in /w/ |

#### O-Series (4)

```
oa oă oe oi
```

| Diphthong | IPA | Type | Example | Notes |
|-----------|-----|------|---------|-------|
| oa | /wa/ | Glide+vowel | hoa, toa, loa | o acts as glide |
| oă | /wa/ | Glide+vowel | xoắn, loắt, hoặc | o glide + ă |
| oe | /wɛ/ | Glide+vowel | khoe, xoe, toe | o glide + e |
| oi | /ɔj/ | Falling | coi, đói, nói | o + i ending |

**Note:** ôi, ơi are oi with tone modifiers on o

| uô | /uə/ | Rising | muốn, cuốn, luôn | u + ô nucleus |
| uy | /uj/ | Falling | quy, huy, tuy | u + y ending |
| ưa | /ɨə/ | Rising | mưa, cửa, lửa | ư + a |

| ươ | /ɨə/ | Rising | mương, người, được | ư + ơ |
| ưu | /ɨw/ | Falling | lưu, hưu, ngưu, cứu | ư + u |

## 5. Spelling Rules Summary

### 5.1 Orthographic Constraints

These rules govern which consonants can precede which vowels:

| Rule | Violation | Correction | Type |
|------|-----------|-----------|------|
| 1 | **c** before e, i, y | Use **k** instead | Velar backness |
| 2 | **k** before a, o, u | Use **c** instead | Velar backness |
| 3 | **g** before e | Use **gh** instead | Velar fricative |
| 4 | **ng** before e, i | Use **ngh** instead | Velar nasal |
| 5 | **gh** before a, o, u | Use **g** instead | Velar fricative |
| 6 | **ngh** before a, o, u | Use **ng** instead | Velar nasal |

### 5.2 Pattern Violations and Examples

**Rule 1: c + {e,i,y}**
- ❌ Invalid: `ce`, `ci`, `cy`
- ✓ Valid: `ke`, `ki`, `ky`

**Rule 2: k + {a,o,u}**
- ❌ Invalid: `ka`, `ko`, `ku`
- ✓ Valid: `ca`, `co`, `cu`

**Rule 3: g + {e}**
- ❌ Invalid: `ge`, `gê`
- ✓ Valid: `ghe`, `ghê`

**Rule 4: ng + {e,i}**
- ❌ Invalid: `nge`, `ngi`
- ✓ Valid: `nghe`, `nghi`

**Rule 5: gh + {a,o,u}**
- ❌ Invalid: `gha`, `gho`, `ghu`
- ✓ Valid: `ga`, `go`, `gu`

**Rule 6: ngh + {a,o,u}**
- ❌ Invalid: `ngha`, `ngho`, `nghu`
- ✓ Valid: `nga`, `ngo`, `ngu`
